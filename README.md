# Weather App
Simple weather app

## Getting started

* Clone repo
* Move `db.json` file to the core directory

## Install JSON Server 

```
npm install -g json-server
```

Start JSON Server

```
json-server db.json
```

## Run

* Open `index.html` in a browser
* To see weather data, go to http://localhost:3000/weather
